## Pygame Zero Environment

### Setup
- Create venv (already created at `.venv`).
- Install deps: `./.venv/Scripts/python -m pip install -r requirements.txt`

### Run
- Windows PowerShell:
  - Activate: `./.venv/Scripts/Activate.ps1`
  - Start game: `python -m pgzrun main.py`

### Notes
- If activation is blocked, run: `Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass`
